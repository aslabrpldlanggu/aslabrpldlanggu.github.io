HTML : PENDAHULUAN

HTML

HTML adalah bahasa markup yang biasa digunakan untuk membuat kerangka sebuah website.HTML sekarang sudah memauki versi ke - 5.

Format File

HTML memiliki format file ".html" ataupun".htm".

TAG,ELEMENT,ATTRIBUTE

Dalam pemrograman,setiap bahasa memiliki syntaxnya masing masing,begitu juga HTML.Dalam HTML biasa disebut Element.Dibawah ini adalah beberapa contoh element:

<p>Ini Element Paragraf</p>

<h1>Ini Element H1 </h1>

<marquee>Ini Element Marquee</marquee>

Sebuah Element HTML terdiri dari TAG,CONTENT mapun ATTRIBUTE.Lalu apa itu TAG,CONTENT,ATTRIBUTE?

<tag attribute="ini-atrribute">Di dalam ini namanya content </tag>

Tag biasanya ditandai seperti ini <p></p>
Sedangkan attribut biasanya bertempat di dalam tag pertama.Gabungan dari ketiga komponen tersebut biasa disebut ELEMENT.

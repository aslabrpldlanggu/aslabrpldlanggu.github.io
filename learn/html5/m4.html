HTML : LIST

List

Penjelasan

  • Untuk membuat sebuah daftar atau list,kita dapat menggunakan tag <ul></ul> dan <ol></ol>
    • <ul></ul> adalah singkatan dari Unordered List.
    • <ol></ol> adalah singkatan dari Ordered List.
  • Setelah membuka Tag tersebut,kita menggunakan <li></li> untuk memberikan isi dari list kita.
    • <li></li> adalah singkatan dari List Item.
  • Kita juga bisa menggunakan <ol></ol> didalam <ul></ul> ataupun sebaliknya seperti contoh kodingan di atas.
  • Tambahan :
    • Tag <br> adalah singkatan dari Break Line yang digunakan untuk membuat barisan baru.

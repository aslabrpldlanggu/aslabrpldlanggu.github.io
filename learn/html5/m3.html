HTML : TABLE

Tabel

Penjelasan

Tabel terdiri dari 2 bagian yaitu baris dan kolom.Baris biasanya diukur dari atas ke bawah,sedangkan Kolom diukur dari kiri ke kanan.

  • Untuk membuat sebuah table kita perlu membuka tag <table></table>
  • Kemudian kita menggunakan tag <thead></thead>,<tbody></tbody>,<tfoot></tfoot> untuk mempermudah dan membedakan bagian bagian dari tabel seperti header,body,dan footer
  • Tag <tr></tr> adalah singkatan dari Table Row yang berarti Baris Tabel.Setiap kita membuat tag tersebut,maka sama saja kita membuat barisan baru dalam suatu tabel.
  • Tag <th></th> merupakan singkatan dari Table Header.Tag ini bisa digunakan untuk mebuat judul judul dari sebuah cell tabel.
  • Tag <td></td> adalah singkatan dari Table Data.Biasnya digunakan untuk membuat sebuah cell data.
  • Setiap Tag <th> <td> memiliki attribute colspan & rowspan
    • Attribute colspan berfungsi untuk menentukan jumlah kolom dalam satu tag <th> <td>
    • Attribute rowspan berfungsi untuk menentukan jumlah baris dalam satu tag <th> <td>

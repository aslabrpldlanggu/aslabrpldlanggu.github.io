HTML : STRUKTUR

Struktur HTML

Seperti gambar di atas,saat kita membuka sebuah tag <p> maka kita harus menutupnya dengan tag </p>

Penjelasan Gambar diatas adalah sebagai berikut :
<!DOCTYPE html> adalah syntax wajib untuk memberi tahu browser bahwa kita menggunakan HTML 5.
Sebelum membuat tampilan element - element yang lain,kita harus membuat element <html></html>
Dalam html terbagi menjadi 2 bagian lagi yaitu tag <head> dan <body>

Didalam TAG HEAD biasanya hanya untuk memasukkan element - element yang tidak ditampilkan di interface.Sedangakan di dalam TAG BODY pada umumnya ditampilkan ke user interface.

Tag Head bisa diibaratkan bumbunya.Sedangkan Tag Body adalah makanannya.

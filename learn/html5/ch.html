CHEATSHEET HTML

Basic Tag
TAG FUNGSI
<html></html> Membuat sebuah Document HTML
<head></head> Tempat untuk mengatur komponen komponen yang tidak akan ditampilkan
<body></body> Tempat untuk mengatur semua komponen yang dapat ditampilkan
<title></title> Untuk memberi nama di title bar,biasanya berada di paling atas sendiri.
Body Attribute
TAG FUNGSI
<body bgcolor=?> Mengatur warna latar belakang dengan hex atau nama warna
<body text=?> Mangatur Warna Text
<body link=?> Mengatur Warna Links
<body vlink=?> Menagtur Warna Link yang telah dikunjungi
<body alink=?> Menagtur Warna Link yang Aktif (saat diklik mouse)
Text Tag
TAG FUNGSI
<pre></pre> Membuat Preformatted Text
<h1></h1> sampai <h6></h6> Membuat judul atau headlines. H1 = Terbesar H6= Terkecil
<b></b> Membuat huruf tebal
<i></i> Membuat huruf italic/miring
<tt></tt> Membuat huruf typewriter
<code></code> Membuat sebuah kode,biasanya menggunkan font monospace
<cite></cite> Membuat kutipan,biasanya tercetak miring
<em></em> Membuat penekanan huruf,biasanya tercetak miring
<address></address> Membuat section alamat,biasanya tercetak miring
<strong></strong> Membuat penekanan huruf,biasanya tercetak tebal
<font size=?></font> Mengatur ukuran text.(Gunakan CSS saja lebih baik)
<font color=?></font> Mengatur warna text.(Gunakan CSS saja lebih baik)
Links Tag
TAG FUNGSI
<a href="URL">ini namanya link</a> Membuat hyperlink menuju ke sebuah URL
<a href="mailto:contoh@gmail.com">Kirim Email</a> Membuat hyperlink menuju ke sebuah alamat email
<a href="#nooby">Ke id nooby</a> Membuat link ke lokasi yang ditentukan(misal #nooby menuju ke element dengan attribute id="nooby")
Formatting Tag
TAG FUNGSI
<p></p> Membuat sebuah paragraph
<br> Membuat baris baru
<blockquote></blockquote> Menyispkan kontent diantara quote
<div></div> Membuat element pembungkus yang berdisplay block
<span></span> Membuat element pembungkus yang berdisplay inline
Lists Tag
TAG FUNGSI
<ul></ul> Membuat unordered list / list yang tak diurutkan
<ol></ol> Membuat ordered list / list yang diurutkan
<li></li> Membuat list item / isi dari ol & ul
<dl></dl> Membuat daftar definisi
<dt></dt> Membuat judur daftar definisi
<dd></dd> Membuat penjelasan dari dt
Graphical Tag
TAG FUNGSI
<hr> Membuat garis horizontal
<hr size=?> Mengatur height/tinggi dari hr
<hr width=?> Mengatur width/panjang dari hr
<hr noshade=?> Membuat garis horizontal tanpa shadow
<img src="URL" /> Menambahkan gambar
<img src="URL" align=? /> Mengatur tata letak gambar
<img src="URL" width=? /> Mengatur width/panjang/besar gambar
<img src="URL" height=? /> Mengatur height/tinggi
<img src="URL" alt=? /> Mengatur text alternatif jikalau gambar tidak bisa dimuat oleh browser

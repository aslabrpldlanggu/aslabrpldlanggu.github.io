Mengenal XAMPP : Pengertian,Fungsi & Cara Installasi.

Pengertian

XAMPP adalah sebuah software webserver yang dikelola oleh Apache.Setiap singkatan XAMPP memiliki arti sendiri.X berarti software XAMPP bisa digunakan di berbagai OS baik Windows,MacOs maupun Linux.A merupakan singkatan dari Apache.M adalah singkatan dari MySQL merupakan salah satu layanan open source database.P adalah singkatan dari PHP.Lalu P berikutnya singkatan dari Perl.

Fungsi

Bayangkan saja,sepeda motor membutuhkan alat utama yang digunakan untuk menjalankannya,yaitu Mesin.Sama seperti website,sebuah website yang dinamis membutuhkan sebuah webserver untuk menghubungkan database dengan sistem ataupun dengan pengguna.XAMPP merupakan webserver tersebut,yaitu yang berfungsi untuk kelancaran jalannya sebuah website yang dinamis.XAMPP juga sebagai alternative webserver offline.

1. Download File 2.Install File 3.Pilih Directori 4.Penggunaan XAMPP

3.Pilih Directori

4.Penggunaan XAMPP

Untuk menggunakan XAMPP,klik 2x pada icon softwarenya.

Untuk mengecek sukses atau tidaknya,kita bisa membuka browser dan mengetikkan "localhost" pada addressbar.

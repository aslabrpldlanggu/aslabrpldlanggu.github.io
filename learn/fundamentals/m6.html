HTML CSS JAVASCRIPT PHP Perbandingan

HTML 5

HTML adalah kepanjangan dari HyperTextMarkupLanguage.HTML bukanlah bahasa pemrograman,melainkan bahasa markup.Kenapa begitu,karena didalam HTML tidak terdapat fungsi fungsi seperti di bahasa - bahasa pemrograman lainnya.Semisal : didalam html kita tidak dapat melakukan penjumlahan,pengurangan,perkalian,pembagian,maupun pengecekan logika.

CSS3

CSS adalah singkatan dari "Cascading Style Sheets".CSS digunakan untuk memperbagus halaman web kita.CSS akan lebih hidup jika dipadukan dengan JAVASCRIPT.

JAVASCRIPT

Javascript adalah bahasa pemrograman yang bisa disebut multifungsi dikarenakan dapat digunakan sebagai bahasa Front End maupun Back End.

PHP

PHP merupakan singkatan dari HypertextPreProcessor.PHP biasa digunakan untuk menghubungkan web static kita ke webserver kita sehingga menjadi dinamis.Tanpa webserver,PHP hanyalah text biasa,jadi kita perlu webserver seperti XAMPP untuk menjalankannya.
